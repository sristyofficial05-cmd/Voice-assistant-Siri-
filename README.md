# 🤖 Mini Siri - AI Student Assistant

Mini Siri is a Python-based AI voice assistant designed especially for students.  
It helps with daily tasks like studying, searching information, taking notes, and more — all through voice commands.

---

## 🎯 Features

- 🎤 Voice Command Recognition
- 🗣️ Female Voice Response
- 📚 Study Assistant (answers questions)
- ⏰ Time & Date Information
- 😂 Jokes & Quotes
- 🔍 Google & 🌐 Web Search
- ▶️ YouTube Playback
- 📖 Wikipedia Integration
- 📝 Notes & To-Do List
- 🧠 Flashcard Quiz
- 📘 Dictionary / Definitions
- 📸 Screenshot Capture
- 🌦️ Weather Updates
- 📰 News Headlines
- ⏳ Alarm & Timer
- ➗ Math Calculator
- 🌍 Translation Support

---
## 🛠️ Tech Stack

- Python 3.13
- SpeechRecognition
- pyttsx3 (Text-to-Speech)
- Tkinter (GUI - Robot Face)
- OpenCV (optional)
- Wikipedia API
- Web APIs (Google, Weather, News)
